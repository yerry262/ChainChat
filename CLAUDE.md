# ChainChat

## Project Overview

A decentralized messaging application built on Ethereum, powered by the XMTP Protocol. Enables secure, peer-to-peer messaging without centralized servers.

## Features

- **Decentralized Messaging**: Built on XMTP Protocol
- **Ethereum Integration**: Uses blockchain for identity and trust
- **No Central Server**: Truly peer-to-peer architecture
- **Privacy-First**: Messages encrypted end-to-end

## Architecture

- **Frontend**: React-based web interface
- **Protocol**: XMTP (Extensible Message Transfer Protocol)
- **Blockchain**: Ethereum for account management
- **Deployment**: GitHub Pages

## Tech Stack

- React
- Ethereum Web3
- XMTP Protocol

## How to Use

1. Connect your Ethereum wallet (MetaMask, etc.)
2. Start messaging other users by their Ethereum address
3. Messages are encrypted and stored on XMTP network
